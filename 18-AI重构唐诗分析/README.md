# AI 重构唐诗：诗人风格与时代脉络分析

## 简介

基于深度学习文本向量化的《全唐诗》风格聚类研究，体验如何使用 AI 分析诗人之间的风格传承关系。

## 学习内容

1. 了解文本向量化技术如何将诗歌转化为数学表示
2. 学会使用 AI 计算诗人之间的风格相似度
3. 掌握聚类分析和降维可视化的基本原理
4. 体验语义检索：用现代白话文搜索古诗
5. 通过词云直观感受不同时代的诗歌特征
6. 探索基于历史文献的诗人社交网络

## 数据来源

| 数据集 | 来源 | 用途 |
|--------|------|------|
| chinese-poetry | GitHub开源项目 | 诗歌文本 |
| CBDB | 哈佛大学/北京大学/中研院 | 诗人生卒年等元数据 |

**繁简转换说明**：原始数据为繁体中文，本实验采用混合方案：繁体原文用于 AI 向量化，简体版本用于界面显示。

## 核心技术

| 技术 | 作用 |
|------|------|
| Sentence-Transformers | 文本向量化（将诗歌转为数学向量） |
| UMAP | 高维数据降维可视化 |
| K-Means | 风格聚类分析 |
| Cosine Similarity | 计算诗人风格相似度 |
| WordCloud | 词云可视化 |
| NetworkX | 诗人社交网络分析 |

## 功能亮点

### 诗人风格相似度
计算任意两位诗人之间的风格相似度，发现李白与杜甫、白居易与元稹等诗人之间的风格关联。

### 语义检索
用现代白话文搜索古诗，例如输入"思念家乡"可找到相关诗句。

### 词云可视化
直观展示不同时代（初唐、盛唐、中唐、晚唐）的高频词汇与主题特征。

### 诗人社交网络
基于 CBDB 历史数据库，可视化诗人之间的社会关系网络。
